# 深色主题 + 响应式 UI 设计

> 日期: 2026-06-07
> 状态: 已批准

## 背景

当前 GUI 使用系统默认 ttk 主题，无自定义样式。窗口 900x650 偏小，Treeview 只显示 5-6 行，控件间距紧凑，整体视觉效果粗糙。

## 设计目标

1. 深色主题：护眼、现代感
2. 大窗口 + 自适应缩放
3. 统一字体和间距

---

## 1. 窗口尺寸

- 默认: `1200x800`
- 最小: `1000x700`
- 启动时居中显示

---

## 2. 配色方案

| 用途 | 颜色 |
|------|------|
| 背景 | `#1e1e2e` |
| 表面（卡片/框架） | `#313244` |
| 边框 | `#45475a` |
| 主文字 | `#cdd6f4` |
| 次要文字 | `#a6adc8` |
| 强调蓝 | `#89b4fa` |
| 成功绿 | `#a6e3a1` |
| 错误红 | `#f38ba8` |
| 警告黄 | `#f9e2af` |
| 选中行背景 | `#45475a` |

---

## 3. ttk.Style 配置

通过 `ttk.Style()` 统一设置所有控件样式：

### 3.1 全局
- 主题: `clam`（最可定制的 ttk 主题）
- 背景: `#1e1e2e`
- 前景: `#cdd6f4`
- 字体: `SF Pro Text 10`（macOS）或 `Helvetica 10`

### 3.2 Treeview
- 背景: `#313244`
- 前景: `#cdd6f4`
- 选中行: 背景 `#45475a`
- 行高: `rowheight=28`
- height: 8 行

### 3.3 LabelFrame
- 背景: `#1e1e2e`
- 标签前景: `#89b4fa`（蓝色强调）

### 3.4 Button
- 背景: `#45475a`
- 前景: `#cdd6f4`
- hover: 背景 `#585b70`
- 活动: 背景 `#89b4fa`，前景 `#1e1e2e`

### 3.5 Entry / Spinbox / Combobox
- 背景: `#313244`
- 前景: `#cdd6f4`
- 插入光标: `#cdd6f4`
- 边框: `#45475a`

### 3.6 Notebook (Tab)
- 背景: `#1e1e2e`
- 选中 Tab: 背景 `#313244`，前景 `#89b4fa`
- 未选中 Tab: 背景 `#1e1e2e`，前景 `#a6adc8`

### 3.7 Text / 日志区
- 背景: `#313244`
- 前景: `#cdd6f4`
- 选中: 背景 `#45475a`

### 3.8 Scrollbar
- 背景: `#313244`
- 滑块: `#45475a`

---

## 4. 布局优化

- Treeview height: 5-6 → 8
- LabelFrame padding: 5 → 8
- 按钮间距: 统一 padx=3, pady=3
- 字体统一:
  - 标题: `("", 12, "bold")`
  - 正文: `("", 10)`
  - 日志: `("Menlo", 10)` (macOS monospace)

---

## 5. 不做的事情

- 不做动画效果
- 不做自定义控件（只用 ttk.Style）
- 不做主题切换功能（只做深色主题）
